# Product Context

This project is a decision-support dashboard for analyzing welfare benefit coverage across Israeli municipalities.

The system helps government analysts:

- detect gaps in benefit utilization
- identify municipalities with unusually low or high uptake
- monitor trends between 2023–2025
- prioritize outreach and policy interventions

The dashboard analyzes:

- 340 municipalities
- 11 benefit types
- population-normalized uptake rates
- gap indicators relative to national averages

Key goals:

1. detect under-utilization of benefits
2. identify extreme trend changes
3. surface insights for policy makers
4. allow drill-down per benefit and municipality

Data is authoritative from the CSV dataset and should not be recalculated in the frontend.

All UI is Hebrew and RTL.

The AI agent should prioritize:

- clarity of visualization
- fast loading
- maintainable React architecture
- minimal complexity
