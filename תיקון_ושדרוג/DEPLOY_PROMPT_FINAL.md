### ערכי Operational_Status:
```
🔴 גבוה מהאשכול   → Gap > +15%
🔵 נמוך מהאשכול   → Gap < -15%
⚪ תואם אשכול     → -15% ≤ Gap ≤ +15%
⚪ ללא סיווג      → חסר אשכול/שיעור
⚪ מדגם קטן       → Small_Pop_Flag = True
```

## תיקון 1: גמלת זקנה ✅ מתוקן בנתונים

**מה תוקן:** שיעור זקנה חושב מחדש מעמודת "מקבלי קצבת זקנה בלבד" (ללא שארים).

**תוצאה:**
- אפס רשויות מעל 100% (לפני: 28 רשויות)
- ממוצע ארצי: ~829 ל-1000
- מקסימום: 975

**מה לעשות בדשבורד:**
- החלף את ה-CSV בקובץ המעודכן
- עדכן שם גמלה מ"קצבת זקנה ושאירים" ל-"קצבת זקנה"
- הוסף tooltip: "שיעור מקבלי קצבת זקנה מתוך אוכלוסיית 65+"
- **ולידציה:** ודא שאף רשות לא מציגה שיעור מעל 100%

## תיקון 2: הוספת 54 מועצות אזוריות

**המצב:** הנתונים כבר כוללים 340 ישויות (286 רשויות + 54 מועצות). ודא שהדשבורד טוען את כולן.

**כללים לתצוגת מועצות:**
```
if Entity_Type == "מועצה אזורית":
  - הצג תג: 🏛️ מועצה אזורית
  - הצג רק Rate_2024 (Rate_2023 ו-Rate_2025 = NaN)
  - הצג הערה: "נתונים זמינים: דצמבר 2024 בלבד"
  - אל תציג: מגמה, שינוי מצטבר, פער מאשכול
  - ב-Dropdown בחירת רשות: הוסף "(מ.א.)" אחרי השם
```
