# דשבורד פערי גמלאות ברשויות

**גרסה:** v1.0 | **תאריך:** מרץ 2026 | **אביעד יצחקי, מינהלי גמלאות, ביטוח לאומי**

---

## תיאור הפרויקט

דשבורד BI לניטור שיעורי גמלאות ברשויות המקומיות בישראל.
הכלי מאפשר לאנליסטים ומנהלים לזהות פערים, מגמות ואשכולות גיאוגרפיים בין רשויות — ולאתר מוקדים הדורשים בחינה מעמיקה.

---

## פיצ׳רים מרכזיים

- **מפת ישראל אינטראקטיבית** — הצגת שיעורי גמלאות לפי רשות עם גוונים לפי עוצמה
- **השוואה לאשכול** — כל רשות מושווית לממוצע אשכולה (1–10) לפי מדד חברתי-כלכלי
- **גרפי מגמה** — שינוי שנתי ומצטבר 2023–2025 לפי סוג גמלה
- **טבלת סינון מתקדמת** — סינון לפי גמלה, אשכול, סטטוס תפעולי ואוכלוסייה
- **Top N מוביל/מוחלש** — רשויות עם הצמיחה/ירידה הגדולה ביותר
- **משוב פיילוט מובנה** — שמירה מקומית, ייצוא ללוח
