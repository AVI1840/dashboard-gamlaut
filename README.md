# דשבורד פערי גמלאות ברשויות

ניתוח פערי גמלאות רווחה ב-280 רשויות מקומיות — נתוני דצמבר 2024.
שלושה מסכים: סקירה כללית, ניתוח לפי גמלה, והשוואת רשויות.

## טכנולוגיה
- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui

## הרצה מקומית
```bash
npm install
npm run dev
```

## בנייה
```bash
npm run build
```

## מבנה תיקיות
```
src/
├── components/
│   ├── dashboard/   # רכיבי דשבורד (גרפים, טבלאות, כרטיסים)
│   └── ui/          # רכיבי shadcn/ui
├── context/         # React Context (סינון סניפים, מצב תצוגה)
├── data/            # נתונים סטטיים (welfareData, flatData)
├── hooks/           # React hooks
├── lib/             # פונקציות עזר
└── pages/           # דפים ראשיים (סקירה, ניתוח גמלה, השוואה)
```
