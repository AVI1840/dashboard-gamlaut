# דשבורד פערי גמלאות ברשויות

ניתוח פערי גמלאות רווחה ב-280 רשויות מקומיות — נתוני דצמבר 2024.
3 מסכים: סקירה כללית, ניתוח לפי גמלה, השוואת רשויות.

## טכנולוגיה
React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui

## הרצה מקומית
```
npm install
npm run dev
```

## בנייה
```
npm run build
```

## מבנה
- `src/components/dashboard/` — קומפוננטות דשבורד
- `src/data/` — נתונים (welfareData, btl_flat_data)
- `src/pages/` — דפים ראשיים
