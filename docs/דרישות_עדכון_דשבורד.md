חישוב כולל גם שארים → שיעור >100% ברשויות מסוימות

### החישוב הנכון:

```
שיעור זקנה = (מקבלי גמלת זקנה בלבד / אוכלוסיית 65+) × 1000
```

### יש להחריג:
- ❌ גמלת שארים
- ❌ עמודות מסכמות ("זקנה ושארים")

### ולידציה:
כל רשות חייבת להיות ≤100% (≤1000 ל-1000 תושבים)
